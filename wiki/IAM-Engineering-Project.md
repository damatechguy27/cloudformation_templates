# IAM Engineering Project

## Overview

This wiki covers comprehensive projects related to AWS Identity and Access Management (IAM) infrastructure, including:

- **Deploying AWS Organizations** - Setting up multi-account AWS environments with organizational structure
- **Configuring Landing Zone** - Establishing a secure, scalable foundation for AWS workloads
- **Setting up AWS IAM Identity Center** - Implementing centralized access management for AWS accounts
- **Creating a test Sandbox AWS account using AWS Organizations** - Provisioning isolated environments for testing and development
- **Integrating AWS Identity Center with Entra ID (Azure AD)** - Enabling single sign-on and identity federation between Azure and AWS
